# Requirements

## 機能要件
- Must:
  - 部屋の写真をアップロードできる
  - AIが「今やるべき1アクション」を1文で生成する
  - 指示カード(1アクション)を表示する
  - カードはスワイプで完了できる
  - 完了後に次のカードが即時に出る
  - 指示対象を示す疑似クロップ(3x3グリッドまたは正規化座標)を提示する
  - 無料プランは1日3カードまで
  - 3枚目完了直後に課金導線を表示する
- Should:
  - 背景に元写真をぼかして表示する
  - 完了時に穏やかなアニメーション/ハプティクスがある
  - 完了履歴を確認できる
  - AIがカード最適化を行う
- Could:
  - 海外向けコピー/ローカライズ
  - 体験を壊さない軽いゲーミフィケーション

## 非機能要件
- 性能:
  - カード切り替えは待ち時間を感じさせない
- セキュリティ:
  - 写真と生成指示は適切に保護される
- 可用性:
  - 主要フロー(写真→カード→スワイプ)は高い成功率で動作する
- 運用性:
  - 生成指示の品質を改善できる運用導線がある

## 制約条件
- 技術:
  - 画像クロップは初期はLLMによる疑似クロップで可
  - 実クロップ処理はクライアントで行う
- 予算:
  - サブスク: ¥500/月 (税込)
  - 海外: $2.99/月
- 期間:
  - MVP優先で段階的に拡張する

## 優先度/段階
- MVP:
  - 写真アップロード
  - 1文カード生成
  - スワイプ完了
  - 次カード即時表示
  - 無料3カード制限と課金導線
- フェーズ2:
  - 完了履歴
  - カード最適化
  - 海外ローカライズ
