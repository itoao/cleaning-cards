# User Journey

## ジャーニー概要
- 判断疲れを減らして「考えずに始められる」体験へ導く

## ステージ別
- 認知:
  - 「片付けを考えずに終わらせる」メッセージに惹かれる
- 興味:
  - スワイプで1アクションずつ進める体験に期待
- 利用開始:
  - 写真をアップロードし、最初のカードが出る
- 継続利用:
  - 短時間で終わる達成感が積み重なる
- 離脱/再訪:
  - 途中で止めても再開しやすいことが記憶に残る

## ペイン/機会
- ペイン:
  - 何から始めるか考える負担
  - 完璧にやらなきゃという圧力
- 機会:
  - 1アクションの提示で開始ハードルを下げる
  - スワイプ完了で短い達成感を積み重ねる
