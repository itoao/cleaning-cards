# Overview

## 背景
- 片付けで最大の障壁は「何から始めるか決められない」「判断で疲れて動けない」こと
- 判断負荷を下げる行動設計で片付け開始を支援する

## 目的
- 考えずに始められて短時間で終われる片付け体験を提供する
- 片付けを「タスク」ではなく「体験」として再定義する

## ゴール/KPI
- ユーザーが「考えずに始められた」と感じる
- 5〜15分で「終わった感覚」を得られる
- 途中離脱しても再開しやすい

## スコープ
- 含む:
  - 写真アップロード
  - AIによる「今やるべき1アクション」の生成
  - 1文指示+該当箇所の画像を含んだカードをスワイプで完了
  - 完了後に次カードを即時提示
  - 無料/有料プランの制限と課金導線
- 含まない:
  - 複数タスクの同時提示
  - チェックリスト型のUI
  - 完璧な掃除の達成

## 前提
- 指示は常に1つのみ
- 1〜2分で終わる行動のみ提示
- 命令口調・説明・理由は書かない
- 句点(。)は使わない

## ステークホルダー
- エンドユーザー
- 企画/プロダクト
- デザイン
- 開発
- 運用/サポート
