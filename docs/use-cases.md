# Use Cases

## 主要ユースケース
- UC-00: 部屋画像を撮影する
  - 開始条件: アプリ起動/カメラ画面表示
  - 成功条件: 部屋全体が写る写真が撮影される
- UC-00a: 撮影後に撮影ガイドが表示される
  - 開始条件: 撮影完了
  - 成功条件: 撮影のコツが短く表示される
  - ガイド例: 「部屋全体が入るように、少し離して撮ってみてください」
- UC-01: 写真をアップロードして最初のカードを受け取る
  - 開始条件: 撮影完了
  - 成功条件: 1枚目のカードが表示される
- UC-02: 指示カードをスワイプで完了する
  - 開始条件: カード表示中
  - 成功条件: 次のカードが即時に表示される
- UC-03: 無料枠の上限到達後に課金導線を表示する
  - 開始条件: 無料3枚目のカードを完了
  - 成功条件: 「今日はここまで。続けますか？」が表示される

## 代替/例外ケース
- EC-01: 撮影が暗い/ブレている
  - 期待動作: 再撮影を促す短いガイドを表示
  - ガイド例: 「明るい場所で、止まってから撮ってみてください」
- EC-02: 写真アップロードに失敗する
  - 期待動作: 再試行導線と簡潔な理由表示
- EC-03: 生成指示が不適切/不明瞭
  - 期待動作: スキップ導線 or 次カード提示
- EC-04: ネットワーク不安定
  - 期待動作: 最低限の再送/待機表現

## ユーザーストーリー
- As a 片付けに疲れている人, I want 何も考えずに次の1アクションが出る so that すぐに動ける
- As a 忙しい人, I want 1〜2分の行動だけ提示される so that すきま時間でも片付けられる
- As a 片付けを続けたい人, I want スワイプで完了できる so that 気持ちよく進められる
